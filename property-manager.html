🔥 ตั้งค่า Firebase เพื่อ Sync ข้ามอุปกรณ์

ทำตาม 3 ขั้นตอนนี้ครั้งเดียว แล้วข้อมูลจะ sync ทุก device อัตโนมัติ

1
ไปที่ console.firebase.google.com → สร้าง Project ใหม่ (ฟรี)
2
เปิด Realtime Database → สร้าง database → เลือก "Start in test mode"
3
ไปที่ Project Settings → เพิ่ม Web App → copy config มาวางในไฟล์ HTML ตรงที่ระบุไว้

แก้ไขในไฟล์ property-manager.html บรรทัด FIREBASE_CONFIG:

const FIREBASE_CONFIG = { apiKey: "AIza...", authDomain: "your-project.firebaseapp.com", databaseURL: "https://your-project-default-rtdb.firebaseio.com", projectId: "your-project-id", storageBucket: "your-project.appspot.com", messagingSenderId: "123456789", appId: "1:123...:web:abc..." };

เมื่อกรอกข้อมูลแล้ว รีเฟรชหน้านี้
